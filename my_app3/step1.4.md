# Завдання 1

Створіть псевдонім p1 команди cd /home/user/practice1 (тут user — ім’я вашого користувача): 

```
alias p1='cd /home/user/practice1' 
```
Протестуйте, що псевдонім працює. Запустіть ще одну копію терміналу та протестуйте псевдонім там, чи працює він зараз? 
Додайте псевдонім до файлу ~/.bashrc вашого користувача. Знову протестуйте псевдонім в новому терміналі, чи працює він тепер ?
Залогіньтесь з-під іншого користувача (ubuntu). Чи працює псевдонім для нього ?
Покращимо псевдонім так, щоб в разі, якщо директорія, до якої звертаються не існує замість повідомлення про помилку команди cd печаталось інше повідомлення:
```
alias p1='cd /home/user/practice1 2> /dev/null || echo "Error! Directory practice1 is not found!"'
```
Зробіть аналогічні псевдоніми, що дозволяють швидко переходити до інших ваших директорії та продемонструйте викладачу.
