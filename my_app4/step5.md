# Завдання 5

Що таке setgid ? За допомогою команди su зайдіть в терміналі в сеанс користувача ubuntu.
Створіть групу shared командою sudo groupadd shared. Додайте до цієї групи користувача ubuntu командою sudo usermod -aG shared ubuntu та вашого користувача (user) командою sudo usermod -aG shared user.
    a) Створіть (використовуючи приставку sudo) директорію /home/shared та встановіть новостворену групу групою власників, а вашого користувача (user) власником для цієї директорії командою chown name:shared /home/shared. Переконайтесь, що для новоствореної директорії встановлено права доступу 775. 
    b) Вийдіть із сеансу користувача ubuntu командою exit. Перейдіть до новоствореної директорії та створіть там файл для запису результатів командою echo ‘without setgid:’ >> res . Потім створіть там файл вашого користувача touch $USER, за допомогою команди su перейдіть в сеанс користувача ubuntu і створіть в тій же директорії файл цього користувача: touch $USER. Подивіться за допомогою команди ls -l, яким власникам та якій групі власників належать файли в директорії /home/shared. Вийдіть із сеансу користувача ubuntu, та запишіть вивід команди ls -l: ls -l >> res.
    c) Видаліть файл вашого користувача. Зайдіть в сеанс користувача  ubuntu  та видаліть його файл в директорії  /home/shared.
    d) Встановіть для директорії /home/shared setgid (права доступу 2775) командою sudo chmod g+s /home/shared.
    e) Перейдіть до новоствореної директорії та створіть там файл користувача ubuntu за допомогою команди touch $USER, за допомогою команди su перейдіть в сеанс вашого користувача і створіть в тій же директорії файл цього користувача: touch $USER. Подивіться за допомогою команди ls -l яким власникам та якій групі власників належать файли в директорії /home/shared та запишіть вивід команди ls -l: ls -l >> res.
Виведіть результати на екран командою cat res та продемонструйте їх викладачу. Видаліть створені в цьому пункті файли та директорії (файли видаляйте від імені їх власників, для видалення директорії зайдіть від імені ubuntu та застосуйте приставку sudo), а також групу командою sudo groupdel shared.
